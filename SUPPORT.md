# Support

Browser Bridge is currently a pre-1.0 Chrome-first extension.

## Before Asking For Help

- Read `README.md`.
- Check `docs/QA.md` for local testing steps.
- Check `docs/PERMISSIONS.md` for permission explanations.
- Check `docs/THREAT_MODEL.md` for security limitations.

## Good Bug Reports Include

- Browser and OS version.
- Extension version.
- Whether the build came from `.output/chrome-mv3` or a release zip.
- Steps to reproduce.
- Expected and actual result.
- Import/export report JSON if relevant, after confirming it contains no secrets.

Do not share archive passwords, decrypted archives, or cookie values.
