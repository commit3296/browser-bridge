# Code Of Conduct

Browser Bridge follows the Contributor Covenant Code of Conduct, version 2.1.

## Our Pledge

We pledge to make participation in this project respectful, harassment-free, and focused on constructive collaboration.

## Expected Behavior

- Use welcoming and inclusive language.
- Respect differing viewpoints and experiences.
- Give and accept constructive feedback.
- Focus on what is best for users and project safety.

## Unacceptable Behavior

- Harassment, insults, or personal attacks.
- Publishing private information without consent.
- Sexualized language or imagery.
- Repeated disruption of project discussions.

## Enforcement

Project maintainers may remove comments, issues, pull requests, or contributions that violate this code. Maintainers may also temporarily or permanently ban contributors for repeated or severe violations.
